### HOW TO SUMMON ME ONTO YOUR MACHINE

**WHAT YOU WILL NEED**

- A terminal
- Python 3.6+
  - You need Python 3. You probably have Python 3. If you don't have Python 3, I can't help you. I'm a rock.
- The willingness to be yelled at by a rock about your nerve levels

```bash
curl -fsSL https://raw.githubusercontent.com/alm108/SinaiTrail/main/install.sh | bash
```

Then type:

```

```
